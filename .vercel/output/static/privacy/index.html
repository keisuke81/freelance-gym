プライバシーポリシー

最終更新日: 2026年3月22日

1. 個人情報の収集について

フリーランスジム（以下「本サイト」）では、ユーザーが入力した情報（請求書の入力内容、計算に使用した数値等）を サーバーに送信・保存することはありません。すべての処理はお使いのブラウザ上で完結します。

一部のツールでは、ユーザーの任意設定により、入力情報をブラウザのローカルストレージに保存することがあります。 この情報は外部に送信されず、ユーザーがブラウザのデータを削除することでいつでも消去できます。

2. アクセス解析について

本サイトでは、サイトの改善を目的として、アクセス解析ツールを使用する場合があります。 これらのツールはCookieを使用し、個人を特定しない形でアクセス情報を収集します。

3. 免責事項

本サイトのツール・記事は参考情報の提供を目的としています。 税務・法律に関する判断は、必ず税理士等の専門家にご確認ください。 本サイトの情報に基づいた行動による損害について、本サイトは責任を負いかねます。

4. お問い合わせ

プライバシーポリシーに関するお問い合わせは、本サイトのAboutページよりご連絡ください。
